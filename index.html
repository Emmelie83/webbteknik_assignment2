Layout

Fråga

  1. Förklara vad CSS-egenskaperna justify-content respektive align-items har för effekt när man använder flexbox för layout.
  2. Vad är en media query och vad används den till?

Svar

1

Egenskapen justify-content styr hur flexobjekt fördelas längs huvudaxeln (vanligtvis horisontellt), medan genskapen align-items bestämmer hur flexobjekt justeras längs den tvärgående axeln (vanligtvis vertikalt). Tillsammans är de mycket användbara för att centrera ett flexobjekt vertikalt och horisontellt, dvs. i mitten av flex-containern.

2

En media query i CSS används för att tillämpa olika stilar beroende på egenskaperna hos den enhet som används, som skärmbredd eller upplösning. Det går därmed att skapa responsiva webbsidor, dvs. webbsidor som anpassar sig till skärmstorleken för olika mobiler, surfplattor och datorer.

Media queries kan dessutom användas för att skapa versioner av webbsidan som passar för utskrift. Exempelvis kan navigationsmenyn och stora bakgrundsbilder ha dolts på sådana sidor eller texten ha getts en font som är bättre läsbar på papper.
